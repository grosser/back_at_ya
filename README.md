# Agent Satisfaction App

## Description:

App that let's agents rate users.

An app which uses the below endpoints:

* /api/v2/users/{{user_id}}/.json

## App location:

* Ticket sidebar

## Set-up/installation instructions:

Download asat.zip, install it, then open a ticket.
